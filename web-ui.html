What to Check in Kong Admin GUI:

  1. Services: Look for "mcp-service" with protocol=grpc
  2. Routes: Look for gRPC route to /mcp.v0.McpService
  3. Consumers: Look for "test-client" consumer
  4. Plugins: Should see:
    • JWT plugin for authentication
    • Rate-limiting plugin (100/min, 1000/hour)
    • OpenTelemetry plugin for tracing
    • Prometheus plugin for metrics

Observability Checks:

  • Prometheus: Query "kong_http_requests_total" to see request metrics
  • Grafana: Create dashboards for Kong metrics and OTEL data
  • Jaeger: Search for "kong-gateway" service to see request traces
